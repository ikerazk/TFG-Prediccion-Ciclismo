# TFG-Prediccion-Ciclismo
Codigo y conjuntos de datos utilizados en el TFG 'Predicción de rendimiento en ciclismo mediante técnicas de Machine Learning'

## Estructura
* 'codigo_TFG.ipynb': Notebook principal con la ejecución de los modelos y experimentos.
* 'Funciones.py': Librería de funciones auxiliares para el notebook principal
* 'race_results_2017_2023.zip': Contiene el archivo race_results_2017_2023.csv, conjunto de datos inicial con el que se ha trabajado a lo largo del proyecto
* 'UCIRiders0519_2x.csv': Archivo utilizado para añadir la variable edad a los datos principales
